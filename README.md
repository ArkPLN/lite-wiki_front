# Lite Wiki

A lite version of Wiki, providing AI-assisted online documents editor and team collaboration application built with React and Vite.


# Main Features

- AI-assisted document editing
- Complete user authentication and authorization
- Team collaboration features
- Markdown Editor and instant preview support by [md-editor-rt](https://github.com/imzbf/md-editor-rt)
- Real-time collaboration

# TechStack

## Frontend
- React
- Vite
- React Router
- React Query
- TailwindCSS
- Lucide React Icons
- Material UI
- md-editor-rt
- Zustand

## Backend
- ASP.NET Core
- Entity Framework Core
- Automapper
- Scalar API
- Semantic Kernel
- PostgreSQL

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `pnpm install`
2. Create the `SERVER_API_URL` in [.env.local](.env.local) to your server API URL
3. Run the app:
   `pnpm run dev`
